Custom Directives
ms-sidenav-helper
Description:

ms-sidenav-helper directive is required if you use sidenavs inside the page content. You can check out the app/main/ui/layouts/simple/left-sidenav/left-sidenav.html file for the simple usage.

Simply, this directive will add a class to the certain elements in order to page and the sidenav act correctly on all devices.
