Custom Directives
ms-masonry
Description:

ms-masonry is a directive that you can use to create Masonry layouts. You can use following options to configure the directive:

vm.masonryOptions = { columnCount : 5, respectItemOrder: false, reLayoutDebounce: 400, responsive : { md: 3, sm: 2, xs: 1 } } Options:
Options Description
columnCount (integer) Column count defines the maximum column count of your masonry layout. ms-masonry will adjust the column count automatically based on responsive options and some other calculations.
respectItemOrder (boolean) Defaults to false to allow classical masonry behaviour, setting it to true enforces that masonry items are kept in order, allowing consistent positioning.
reLayoutDebounce (integer) Debounce amount in milliseconds before reLayout can happen. This will prevent reLayout action from happening too much if you are resizing your screen or the device rotation is being changed.
responsive (object) You can define column counts for individual Angular Material media step definitions.
