Custom Directives
ms-random-class
 Description:

ms-random-class directive can be used for adding randomly selected class from an array. It's used mainly for generating random background colors in the template, but you can use it anywhere if you find it useful.

Attribute Description
ms-random-class (Array) This attribute must be present in order to ms-random-class directive to work.
Example: vm.colors = [ 'yellow', 'red', 'blue', 'orange', 'green' ]
