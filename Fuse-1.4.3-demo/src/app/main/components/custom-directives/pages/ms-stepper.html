Custom Directives
ms-stepper
Description:

ms-stepper is a form step directive that follows Google's Material Design specifications. It allows you to create stepped forms. Unlike ms-form-wizard, this directive doesn't use any other material elements, therefore it's a lot more flexible and easy to use:

Vertical Stepper:
ms-*-stepper-step

This directive requires ms-*-stepper directive and it's used for creating individual steps. You can either directly wrap your forms with this directive, or you can use ng-include to include them.

ms-stepper-step Attributes
Attribute Description
ng-form (string) This attribute/directive is required for steps to work correctly. It requires a unique, string value. You can think this as the unique name of the step.
step [Optional] (integer) Optional value for the step number. If you want to manually order your steps, you can use this attribute.
step-title (string) Step title that will show up in the step navigation.
optional-step [Optional] If given, this attribute will make the step optional. Therefore, that step can be passed through without filling its input fields and it will not be included into the main form validity check.
