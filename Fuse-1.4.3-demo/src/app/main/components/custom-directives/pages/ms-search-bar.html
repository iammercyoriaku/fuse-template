Custom Directives
ms-search-bar
Description:

ms-search-bar is an autocomplete supported search bar directive. It has various options and events for you to use.

Options:
Options Description
debounce (integer) Model update debounce. Can be used to stop triggering the on-search event for every keystroke.
on-search (function (query)) A callback function that will trigger on model update. Must return an Array or a Promise object that caries an Array. Has 'query' parameter to access the search query.
on-result-click (function (item)) A callback function that will trigger on search result click. Has 'item' parameter to access the clicked item.
on-expand (function) A callback function that will trigger when search bar expanded.
on-collapse (function) A callback function that will trigger when search bar collapsed.
