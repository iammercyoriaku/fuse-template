Custom Directives
ms-widget
Description:

ms-widget directive is a helper directive for creating widgets. When it used with helper classes such as padding, margin, height helpers, it's very easy to create custom widgets.

Attribute Description
flippable If it's true, widget will be flippable. If you don't want to make the widget flippable, you can make this attribute false and use only <ms-widget-front></ms-widget-front> wrapper to show widget content.
